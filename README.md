### Gemini API 怎么用

**第一步：获取 API Key**

有两种方式：

**方式 A：官方 API**

- 访问 Google AI Studio
- 创建 API Key
- 需要魔法上网和外部支付方式

**方式 B：第三方平台导航**

- 使用中文使用指南和镜像导航站了解可选方案
- 使用前先确认平台的稳定性、计费方式和接口兼容性

推荐参考：

- `https://gemini-chinese-guide.com`
- `https://gemini-mirrors.com`

**第二步：配置代码**

以 Python 为例：

```python
import google.generativeai as genai

# 配置 API Key
genai.configure(api_key='your-api-key')

# 创建模型
model = genai.GenerativeModel('gemini-3.1-pro')

# 生成内容
response = model.generate_content('Hello!')
print(response.text)
```

**第三步：调用 API**

在代码中调用，就像调用函数一样简单。

## Gemini 常见问题

**Q1：Gemini 镜像站安全吗？**

A：使用任何第三方平台前，都建议先评估其隐私政策、服务稳定性和数据处理方式。

**Q2：Gemini 镜像站会一直免费吗？**

A：免费政策通常会调整，建议以平台最新说明为准。

**Q3：Gemini 镜像站和 Gemini 官方有什么区别？**

A：通常是访问方式、计费方式和服务主体不同，具体能力与限制以平台实际提供为准。

**Q4：Gemini 可以同时用多种方式吗？**

A：可以。我自己就是镜像站 + API 一起用。

**Q5：哪种 Gemini 使用方式最省钱？**

A：轻度使用时，镜像站或 AI Studio 通常更省钱。

**Q6：哪种 Gemini 使用方式最稳定？**

A：官方订阅通常最稳定，但需要付费。

**Q7：开发者使用 Gemini 应该选哪种？**

A：如果只是测试，用镜像站或 AI Studio；如果要集成到应用，用 API。

**Q8：Gemini 镜像站支持所有 Gemini 模型吗？**

A：具体支持情况要以平台当前展示的模型列表为准。

**Q9：支持 Gemini 的 ChatGPT Plus 镜像站和 Gemini 镜像站有什么区别？**

A：前者通常支持多个 AI 模型，后者更聚焦 Gemini 系列。

**Q10：Gemini AI Studio 的免费配额够用吗？**

A：对于轻度使用和测试通常够用，重度使用建议选择其他方式。
